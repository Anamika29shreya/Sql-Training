HTML Entity Example

Advantage of using an entity name: An entity name is easy to remember. Disadvantage of using an entity name: Browsers may not support all entity names, but the support for entity numbers is good.

Note: Entity names are case sensitive.

A space character:  

The less-than sign: <

The greater-than sign: >

The ampersand (and) sign: &

The cent sign: ¢

The pound sign: £

HTML Special Character Example

An a with a grave accent: à

An o with a grave accent: ò

An o with a circumflex accent: ô

An o with a tilde: õ
